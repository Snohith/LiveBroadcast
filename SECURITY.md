# Security Policy

## Supported Versions

The following versions of Live Cricket Score API are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an email to the maintainers or report it privately via GitHub Security Advisories. 

Please **do not** open public issues for security vulnerabilities.

All security vulnerabilities will be promptly addressed. Thank you for helping keep this project secure!
